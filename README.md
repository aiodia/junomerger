# Jupyter Notebook Merger

Simple library to merger several jupyter notebook into one


## Requirements

- Python 3.6
- Numpy
- os
- io
- bs4
- urllib

## Getting Started

### Installing

install packages :
	pip install git+git://github.com/aiodia/junomerger.git

### Usage

see examples folder for usage
